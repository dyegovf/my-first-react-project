# 🚀 My First React Projects

This repository contains my journey learning **React.js**, organized into multiple mini-projects.  
Each folder represents a specific concept or skill I practiced, from the basics of React to more advanced topics like Context API and theme switching.

---

## 📂 Projects & Learnings

### 01basicreact
- **Learned:** React basics — JSX, functional components, props.
- **Concepts:** Component creation, rendering, passing props.

### 01basicvite
- **Learned:** Project setup with Vite.
- **Concepts:** Fast development environment, hot reload, project structure.

### 02counter
- **Learned:** State management with `useState`.
- **Concepts:** Event handling (`onClick`), updating state, reactivity.

### 03tailwindprops
- **Learned:** Styling with Tailwind CSS and props.
- **Concepts:** Utility-first CSS, dynamic styling via props.

### 04bgChanger
- **Learned:** Changing background color dynamically.
- **Concepts:** `useState` applied to styles, interactive UI.

### 05passwordgenerator
- **Learned:** Generating random passwords.
- **Concepts:** `useState`, `useEffect`, handling inputs, string manipulation.

### 06currencyconverter
- **Learned:** Fetching and using external APIs.
- **Concepts:** `useEffect`, `fetch`, async data handling, updating UI.

### 07reactRouter
- **Learned:** Navigation with React Router.
- **Concepts:** `BrowserRouter`, `Routes`, `Link`, dynamic routing.

### 08miniContextApi
- **Learned:** Introduction to Context API.
- **Concepts:** `createContext`, `useContext`, `Provider`, global state sharing.

### 09themeswitcher
- **Learned:** Dark/Light theme switching.
- **Concepts:**  
  - React Context API (`createContext`, `useContext`, `Provider`).  
  - Custom Hook (`useTheme`) for consuming context.  
  - Tailwind CSS dark mode (`dark:` classes).  
  - **Pattern:** Context with default values + custom hook (recognized best practice).  

### 10todocontextLocal
- **Learned:** Combining Context API with localStorage persistence.
- **Concepts:**  
  - Managing a **Todo list** with global state.  
  - Persisting data in **localStorage** so tasks survive page reloads.  
  - Advanced Context usage: state + actions in provider.  
  - Practical example of scalable state management.

### 11blogapp
- **Learned:** Building a larger project with external backend (Appwrite).
- **Concepts:**  
  - Full CRUD operations (create, read, update, delete) with a backend service.  
  - Authentication and database integration using **Appwrite**.  
  - Structuring a **React mega project** with multiple pages and features.  
  - Applying all previous concepts (routing, context, hooks, API calls) in a real-world app.

### customReact
- **Learned:** Custom experiments with React.
- **Concepts:** Creating custom hooks, reusable components, project organization.

---

## 🔑 Key Concepts Consolidated
- **React Basics:** JSX, props, components.  
- **State Management:** `useState`, `useEffect`.  
- **Styling:** Tailwind CSS, conditional classes.  
- **Routing:** React Router.  
- **Global State:** Context API + custom hooks.  
- **Persistence:** localStorage integration.  
- **Dark Mode:** React + Tailwind integration.  
- **Backend Integration:** Appwrite for authentication and CRUD.  
- **Patterns:** Context with default values, custom hook consumption.
